### Fahren 

#### Start

Wenn der Bulli kalt ist, geht er gerne mal aus wenn man kein Gas gibt. Das legt sich nach ~10 Minuten Fahrt oder Motorlaufenlassen im Stand.

#### Tanken

Der Bulli tankt `super`. Kein `E10`!
